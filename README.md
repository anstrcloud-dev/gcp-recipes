# GCP Recipes

Recipe management app built with Flask, deployed on GCP Cloud Run with Firestore, Secret Manager, CI/CD via GitHub Actions and Terraform.

## Tech Stack
- **App**: Python + Flask
- **Database**: Firestore
- **Hosting**: Cloud Run
- **Containers**: Docker + Artifact Registry
- **IaC**: Terraform
- **CI/CD**: GitHub Actions
- **Secrets**: Secret Manager

## Features
- Add, browse, search and delete recipes
- Search by ingredient or category
